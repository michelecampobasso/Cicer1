# cicer1

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).

> NB

The root folder of this project should be placed in your root folder in your local machine (run command "cd"). Otherwise npm run build will fail.
